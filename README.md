# mybatis-board

## 🖥️ 프로젝트 소개
MyBatis 를 이용한 Springboot 게시판 사이트
3일 안에 끝내기 도전!

## 개발 기간
23.11.29 ~ 23.12.01

## ⚙️ 개발 환경
- **Language** : 'Java 17'
- **Farmework** : 'Springboot 3.1.6'
- **Database** : 'h2 database 2.2.224'
- **ORM** : 'MyBatis 3.0.1'
- 'thymeleaf 3.1.6'
- 'lombok

## 📌 주요 기능
#### 회원관리
- 로그인
- 회원가입
- ID, PW 찾기
- 마이 페이지 (회원정보 변경 or 탈퇴)

#### 게시판
- 글 목록
- 글 상세 페이지
- 글 작성, 읽기, 수정, 삭제 (CRUD)

## 목표

- 겉으로 보이는 페이지의 모양보단 'MyBatis' 를 통해 'Database' 와 'Server' 를 접근하여 기능 구현하는데 집중
